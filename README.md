# pgEdge Documentation Skill

A Claude Code plugin that provides pgEdge documentation writing
guidelines, style rules, and MkDocs project scaffolding.

## Installation

Install the plugin with the following commands in Claude Code:

```
/plugin marketplace add pgEdge/pgedge-docs-skill
/plugin install pgedge-docs-skill@pgedge-plugins
```

## Features

The plugin provides a skill that:

- enforces pgEdge documentation style and formatting conventions.
- scaffolds new MkDocs Material projects with pgEdge branding.
- guides README structure, changelog format, and navigation layout.
- validates documentation against a comprehensive review checklist.

## Usage

The skill activates automatically when you work on documentation
tasks such as writing, editing, reviewing, or structuring pgEdge
docs. You can also invoke it explicitly:

```
/pgedge-docs-skill:pgedge-docs
```

### Scaffolding a New Project

Ask Claude Code to set up documentation for a new project; the skill
includes a complete MkDocs Material template with pgEdge branding,
light/dark mode support, and starter files.
